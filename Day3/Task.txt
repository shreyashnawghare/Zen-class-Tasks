Read about the difference between window, screen and document in javascript:-
Window:-
The window is the first thing that gets loaded into the browser. This window object has the 
majority of the properties like length, innerWidth, innerHeight, name, if it has been closed, 
its parents, and more.

Screen:-
window.screen or just screen is a small information object about physical screen dimensions.

Document:-
Each window object has a document object to be rendered.
